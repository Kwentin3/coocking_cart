# Data statuses

Используй статусы данных для важных значений:

- Подтверждено пользователем: пользователь явно подтвердил значение в диалоге.
- Рассчитано системой: значение получено из введенных данных по простой расчетной логике.
- Взято из справочника: значение связано с указанным справочным источником.
- Требует проверки: значение нужно подтвердить ответственным лицом предприятия.
- Требует производственной проработки: значение зависит от фактического приготовления, выхода, потерь, времени, температуры или упаковки.
- Не заполнено: поле нужно для документа, но данных пока нет.

Применяй статусы к выходу порции, количеству порций, брутто/нетто, потерям, БЖУ, срокам хранения, температурным режимам, условиям доставки и требованиям безопасности.
